Brain-Score Tutorial

The Brain-Score platform allows users to score models on public data via the command line on your machine, once installed. It also allows scoring models on all data (public and private) via the website. In the section we will cover how to use the CLI to score a model on public data.

Step 1: Install Necessary Packages

In order to use Brain-Score on your machine, you need to install it. Luckily, we have tried to drastically simplify this process (we strongly recommend setting up a Virtual Environment for all your Brain-Score projects, but this is not required). Run the following commands in the command line of your choosing (example is for Unix-based machines): 



git clone https://github.com/brain-score/brain-score.git
cd brain-score
python -m pip install -e "."

This will pull the most recent copy of Brain-Score into your local machine and install all the necessary packages.

Step 2: Run a model on a benchmark (via CLI)

Next, make sure your working directory is /brain-score (you can confirm with a pwd call), and run the following command to score a model (pixels) on a sample benchmark’s publicly available data (dicarlo.MajajHong2015.IT-pls) 



python3 brainscore_vision  score --model_identifier='alexnet' --benchmark_identifier='dicarlo.MajajHong2015.IT-pls'

You should get a message like below, indicating what the score is: 



xarray.Score (aggregation: 2)
array([0.01538053, 0.00571393])
Coordinates:
  * aggregation  (aggregation) U6 'center' 'error'
Attributes:
    raw:                   xarray.Score (aggregation: 2)>\narray([0.10192326...
    ceiling:               xarray.Score (aggregation: 2)>\narray([8.21840601...
    model_identifier:      pixels
    benchmark_identifier:  dicarlo.MajajHong2015.IT-pls

Process finished with exit code 0

Let’s break down what these numbers mean. First off, your score is 0.0153, the first number in the first xarray listed, along with the error, which is 0.00571. Next, you can see two other attributes: raw, which represents the unceiled score (0.1019) of your model, and the ceiling (8.28*10-2), which is the highest score a model can get. Your final score above thus is a simple calculation of raw/ceiling. Note: This example scores a model on a sample benchmark's data in Brain-Score Vision, but you can follow a similar procedure for Brain-Score Language with the following command. 



python brainscore_language score --model_identifier='distilgpt2' --benchmark_identifier='Futrell2018-pearsonr'

Please note that compute times may vary; running on a 2021 Macbook Pro M1 Max takes about 10 minutes. If you would like to learn how to submit a model to the website, please visit Deepdive 1.
