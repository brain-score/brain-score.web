Welcome, Benchmarkers!

Brain-Score uses plugins as its main way of interacting with submissions. These plugins can be models, benchmarks, data, or metrics. To create a full benchmark and have it on the website, most of the time 3 plugins are required: a benchmark plugin (which contains the code on how the model is to be evaluated), a data plugin (which

Packaging data involves wrapping up both stimuli and subject data and uploading them for models to access. Once this is done, then you can create the benchmark itself, which involves setting benchmark hyperparameters, defining a metric, and defining a ceiling, amongst other things.

To package your data and create the benchmark, please visit the corresponding guides below. You will need both your stimuli and your subject data (.csv format or similar). These guides cover both neural and behavioral benchmarks.
